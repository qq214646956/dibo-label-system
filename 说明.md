# 标签打印系统

基于 qrlayout 的中文版条码标签设计器，支持拖拽式标签模板编辑、条码/二维码生成、批量导出。

---

## 在另一台电脑上使用

### 前提条件

- 安装 [Node.js](https://nodejs.org)（版本 ≥ 16）

### 步骤

1. 将整个 `标签打印系统` 文件夹复制到目标电脑
2. 打开命令行（CMD 或 PowerShell），进入 react-demo 目录：

```bash
cd 标签打印系统\qrlayout\examples\react-demo
```

3. 安装依赖（仅第一次需要，会自动安装中文翻译）：

```bash
npm install
```

4. 启动：

```bash
npm run dev
```

5. 浏览器打开终端显示的地址（通常是 http://localhost:5173）
6. 点击「标签」→「新建标签」→ 进入拖拽式标签设计器（全中文界面）

---

## 功能说明

| 功能 | 说明 |
|------|------|
| 拖拽式标签设计 | 可视化拖拽文本、二维码到标签画布上 |
| 自定义标签尺寸 | 支持自定义任意尺寸（如 100×70mm） |
| 二维码/条码 | 支持 Code128、QR 码，动态数据绑定（{{变量名}}） |
| 多格式导出 | PNG / PDF / ZPL（Zebra 热敏打印机） |
| 批量打印 | 勾选多条数据 → 一键批量导出 |
| 数据管理 | 内置员工/设备/库位数据管理 |

## 目录结构

```
标签打印系统/
├── 说明.md
└── qrlayout/
    ├── packages/
    │   ├── core/          # 核心引擎（条码渲染、PDF/ZPL 导出）
    │   └── ui/            # 拖拽式设计器 UI（中文版，dist/ 已预构建）
    └── examples/
        └── react-demo/    # 主应用入口
            ├── setup-chinese.js   # 中文安装脚本（npm install 自动执行）
            └── package.json       # 含 postinstall 钩子
```

---

## 注意事项

- 首次 `npm install` 会自动运行中文安装脚本
- 标签数据存储在浏览器 localStorage，不会上传到服务器
- 包大小约 8.4MB（不含 node_modules，由 npm install 自动生成）
