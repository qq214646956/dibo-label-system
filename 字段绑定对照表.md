# 字段绑定对照表

> 来源：SAP RFC `ZFM_ZSDELIVERY_DETAILS` 出货主数据  
> 用法：在标签设计器中用 `{{字段名}}` 引用，渲染时自动替换为实际值

## 基础字段

| 绑定语法 | 字段名 | 描述 |
|----------|--------|------|
| `{{VBELN}}` | VBELN | 交货单号 |
| `{{POSNR}}` | POSNR | 交货单行号 |
| `{{VGBEL}}` | VGBEL | 销售凭证 |
| `{{VGPOS}}` | VGPOS | 销售凭证项目号 |
| `{{WADAT_IST}}` | WADAT_IST | 过账日期 |
| `{{ERDAT}}` | ERDAT | 创建日期 |
| `{{MATNR}}` | MATNR | 物料编码 |
| `{{MAKTX}}` | MAKTX | 物料描述 |
| `{{PM}}` | PM | 品名 |
| `{{GROES}}` | GROES | 规格型号 |
| `{{KDMAT}}` | KDMAT | 客户物料 |
| `{{ARKTX}}` | ARKTX | 销售订单项目短文本 |
| `{{LFIMG}}` | LFIMG | 实际已交系统重量 |
| `{{LFIMG_HY}}` | LFIMG_HY | 实际已交行业重量 |
| `{{MEINS}}` | MEINS | 单位 |
| `{{CHARG}}` | CHARG | 批次编号 |
| `{{ZFZSL}}` | ZFZSL | 辅助数量 |
| `{{ZFZDW}}` | ZFZDW | 辅助单位 |
| `{{ZBZCD}}` | ZBZCD | 标长(M) |
| `{{ZBZKD}}` | ZBZKD | 标宽（MM） |
| `{{ZBZHD}}` | ZBZHD | 标厚（MM） |
| `{{ZBZBZ}}` | ZBZBZ | 比重 |
| `{{ZBZZL}}` | ZBZZL | 标重 |
| `{{ZHYZL}}` | ZHYZL | 行重 |
| `{{ZBZCD2}}` | ZBZCD2 | 行业长度(M) |
| `{{ZBZKD2}}` | ZBZKD2 | 行业宽度（MM） |
| `{{KUNNR}}` | KUNNR | 客户编码 |
| `{{NAME1}}` | NAME1 | 客户名称 |
| `{{SORTL}}` | SORTL | 客户简称 |
| `{{USERNAME}}` | USERNAME | 过账人 |
| `{{EX_TEXT}}` | EX_TEXT | 销售订单抬头文本 |

## 打印辅助字段

| 绑定语法 | 描述 |
|----------|------|
| `{{DATE}}` | 打印日期 |
| `{{TIME}}` | 打印时间 |
| `{{DATETIME}}` | 打印日期时间 |

## 特殊函数

| 语法 | 参数 | 说明 | 示例 |
|------|------|------|------|
| `{{_SEQ:start,step,digits}}` | 起始值, 步长, 位数 | 批量打印的序号 | `{{_SEQ:1,1,3}}` → 001, 002, 003 |
| `{{_RANDOM:min,max,dec,step}}` | 最小值, 最大值, 小数位, 步长 | 随机数 | `{{_RANDOM:10,50,0,5}}` → 15, 30, 45… |

> 在标签设计器右侧属性面板中，输入框下方会自动显示可点击添加的字段按钮。
